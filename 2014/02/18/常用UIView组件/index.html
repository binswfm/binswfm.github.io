UIButton

  • 常用属性

1.Type

+ Custom:选择此属性外观主要依靠自定义实现。
+ System:系统能够默认风格。
+ Detail Disclosure:详情标示。
+ Info Light:显示“i”图标的图形按钮。
+ IndoDark:显示“i”图标的图形按钮。
+ Add Contact:显示黑色“+”图标图形按钮。
  1. State Config
    • 按钮的状态
      • 默认状态（Default）
      • 高亮状态（Highlighted）
      • 选中状态（Selected）
      • 禁用状态（Disable）
  2. Title
    • 可以选择Plain和Attributed文本方式
    • 制定显示的字符信息
  3. Font
    • 文本字体、大小、字体风格
  4. Text Color
    • 文本的颜色
  5. Shadow Color
    • 阴影颜色
  6. Image
    • 设置为图片按钮，Title属性将不会起作用。
  7. Background
    • 设置背景图片
  8. Shadow Offset
    • 文本和阴影的偏移量
  9. Link Break
    • 省略方式
  10. Edge
    • Content:内容作为边界
    • Title:文本作为边界
    • Image:图片作为边界
  11. Inset
    • 边界距离
