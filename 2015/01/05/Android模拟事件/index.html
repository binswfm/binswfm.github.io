  • 命令 input [] […]
    • source 指定输入设备
      • trackball 轨迹球
      • joystick 操控杆
      • touchnavigation
      • mouse
      • keyboard
      • gamepad
      • touchpad
      • dpad
      • stylus
      • touchscreen
    • command 指定动作
      • text (默认设备touchscreen)
      • keyevent [—longpress] … (默认设备keyboard)
      • tap (默认设备touchscreen)
      • swipe [duration(ms)] (默认设备touchscreen)
      • press (默认设备trackball)
      • roll (默认设备trackball)
      • tmode
